\begin{frame}
\frametitle{Crash course in Type Theory}
Types:
 \begin{itemize}[<+->]
\item $\bn{0},~\bn{1},~\bn2,~\ldots,\NN$\quad inductive types
\item $\prod_{x:A}B(x),\ \sum_{x:A}B(x)$\quad dependent types (nondep: $\to,\times$)
\item $\UU_0 : \UU_1 : \UU_2 : \cdots$\quad universe types
\item $\Id_A(a,a')$\quad identity with constructors $\refl_a:\Id_A(a,a)$
\end{itemize}
%\vspace*{1cm}               
Useful:
\begin{itemize}[<+->]
\item $A\to(B\to C)$,\quad higher-order function type
\item $\vectype n A$,\quad type of vectors of $n$ elements of $A$
\item $\bigl(A\mapsto (x \mapsto x)\bigr):\prod_{A:\UU_\ell}(A\to A):\UU_{\ell+1}$,\quad polymorphism
\item NB One can abstract from $\ell$ also: universe polymorphism 
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Two great ideas}
\begin{itemize}[<+->]
\item Propositions-as-types (Curry, Howard, de Bruin): 
(dependent) types can represent propositions
in logic, with $\to,\times,\prod$ for $\to,\land,\forall$. 
With some effort one also gets $\neg,\lor,\exists$.
\begin{itemize}[<+->]
\item A computational interpretation of proofs
\item Proof assistants: (AUT,MLTT), Coq(Rocq), Agda, Lean, ...
\end{itemize}
\item Tower of $\Id$-types: $p,p':\Id_A(a,a')$; 
$s,s':\Id_{\Id_A(a,a')}(p,p')$; etc.
\begin{itemize}[<+->]
\item a homotopy interpretation of equality (Voevodsky, Awodey, Warren)
\item HoTT, HITs, synthetic homotopy theory, univalent mathematics
\end{itemize} 
\end{itemize}
\vspace*{1cm}
Notation: $a=_A a'$ for $\Id_A(a,a')$, subscript $A$ may be omitted.\\
Terminology: and element of $a= a'$ is called a \emph{path} from $a$ to $a'$.  
\end{frame}


\begin{frame}%[fragile]
\frametitle{Homotopy levels}
\begin{itemize}[<+->]
\item Cumulative and non-exhaustive hierarchy of types,
based on the complexity of the identity types.
\item Proposition: a type in which all elements are equal
\item Set: a type in which all identity types are propositions
\item Groupoid: a type in which all identity types are sets
\item and so on
\end{itemize}
Examples (often you need something extra to prove it):
\begin{itemize}[<+->]
\item Propositions: $\bn{0},~\bn{1}~(\text{not}~\bn{2})$, 
$\sum_{x:T} t=x$, $T\to\bn0$ \quad(any $T$).
\item Sets: any prop, $\NN$, $T\to\NN$, $\prod_{n:\NN} \vectype n S$ 
\quad(any $T$, set $S$)
\item Groupoid: any set, and now what?  Univalence!
\end{itemize}

\end{frame}

\begin{frame}
\frametitle{Univalence Axiom for (h-)sets}
\begin{itemize}[<+->]
\item $\UA_{\Set}$: for all sets $S,S'$, $S=S'$ is a set, and the map
$$i: (S=S') \to (S\simeq S'): \refl_S\mapsto \id_S$$
is a bijection, $S\simeq S'$ is the type of bijections $S\to S'$.
\item Assuming $\UA_{\Set}$,
\begin{itemize}[<+->]
\item $\Set$ is a groupoid and not a set: $(\bn2=\bn2)=\bn2$
\item $i$ identifies $S=S$ with the set of permutations of $S$
\item the action of $i$ on paths equips $S=S$ with a group structure.
\item Elements of $S=S$ are called \emph{symmetries} of $S$ in $\Set$.
\end{itemize}
\item $\UA_\UU$: for all $A,B:\UU$, the map
$$i: (A=B) \to (A\simeq B): \refl_A\mapsto \id_A$$ is an equivalence,
where $A\simeq B$ is the type of equivalences $A\to B$.\footnote{%
The notion of \emph{equivalence} generalizes that of a bijection of sets 
to types. Given types $X,Y$, a map $f:X\to Y$ is an equivalence if for 
all $y:Y$ the inverse image (fiber) 
$f^{-1}(y)\defeq\sum_{x:X}(y=f(x)$ is a true proposition
(a \emph{contractible} type), that is, a proposition such that 
we have an element (proof, center).}
\end{itemize}
\vspace*{1cm}          
\end{frame}

\begin{frame}
\frametitle{Univalent group theory}
 \begin{itemize}[<+->]
\item Group $G$: pointed connected\footnote{%
A type $T$ is \emph{connected} if it is non-empty and for every
$x,y:T$ there exists (weaker than `we have') a path $p:x =y$
($\Trunc{x=y}$, explain). Notation: $T_{(t)} \defeq \sum_{x:T}\Trunc{t=x}$, 
the connected component of $T$ at $t:T$.}
groupoid $(BG,\sh_G)$
\item Underlying set of $G$ is $UG \defeq (\sh_G  = \sh_G)$ (`symmetries')
\item Group operation on $UG$ is path composition, unit is $\refl_{\sh_G}$
\item Aim: develop group theory on the level of the pointed groupoids,
called `classifying types'.
\item Examples of univalent groups
 \begin{itemize}[<+->]
\item $(\Set_{(S)},(S,!))$, the symmetric group of $S:\Set$ (we omit "!")
\item $((\sum_{X:\Set}(X\simeq X))_{(\Zet,\Ss)},(\Zet,\Ss))$, 
the group $\ZZ$ of the integers
\item $((\sum_{X:\Set}(X\simeq X))_{(\mathbf{m},\Ss_m)},(\mathbf{m},\Ss))$,
the cyclic group $C_m$ %, $m>0$
\item $((\sum_{X:\Set}(X\simeq X))_{(\mathbf{4},\pi)},(\mathbf{4},\pi))$ 
with $\pi=(01)(23)$, ?
\end{itemize}
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Univalent group theory (ctnd)}
 \begin{itemize}[<+->]
\item Homomorphism: $f: (BG,\sh_G)\ptdto(BH,\sh_H)$,\\
action on symmetries $(g:BG)\mapsto f_\pt^{-1}\cdot \ap{f}(g)\cdot f_\pt$
\item $G$-set: $X: BG\to\Set$, indeed a homomorphism 
$$(X,\refl_{X(\sh_G)}): (BG,\sh_G)\ptdto(\Set_{(X(\sh_G))},X(\sh_G))$$
\item Two equivalent notions of a subgroup of a group $G$:
\begin{itemize}[<+->]
\item $((BH,\sh_H),m)$ with $m:(BH,\sh_H)\ptdto (BG,\sh_G)$ a monomorphism
(set bundle, or injection on symmetries)
\item $(X,x)$ with $X$ a $G$-set, $\sum_{z:BG} X(z)$ connected, and $x:X(\sh_G)$
\end{itemize} 
\item Now, what group is the subgroup in each case?
\begin{itemize}[<+->]
\item $\im(m)\defeq\sum_{z:BG}m^{-1}(z)\simeq\sum_{z:BG,w:BH}(z=m(w))\simeq BH$\\
(as $m$ mono, $m^{-1}(z)$ is a set),
with shape $\sh_H$\footnote{Good or bad: $(BH,\sh_H)$ doesn't depend on $m$?}
\item $(\sum_{z:BG} X(z), (\sh_G,x))$, symmetries of $\sh_G$ stabilizing $x$\\
(picture)
\end{itemize} 
\end{itemize}
\end{frame}

Setting:  HoTT ≈ MLTT + univalence + some HITs (propositional truncation).

1. Group G is given by a pointed connected 1-type, BG, and a point sh_G (the shape).
Underlying set of group G: UG :≡ (sh_G  = sh_G), the set of symmetries of the shape.
Group operation: path composition.
BUT: we develop group theory as much as possible on the level of the pointed 1-types.

2. Examples: (S^1 , base); (Set_(S) , S), eg. S={0,..,n}; ((ΣS:Set (S=S))_ , (Z,s));
((ΣS:Set (S=S))_ , ({0,..,n-1},s mod n)).

3. Homomorphism is given by a pointed map (f,p) : (BG,sh_G) → (BH,sh_H).
Monomorphism is a homomorphism such that f is a set bundle (ap_f  is injective).
Subgroup H of G is given by a monomorphism (m,p) : (BH,sh_H) → (BG,sh_G).

4. Example: c: (1,*) → (S^1 , base), c(*):≡base, gives trivial subgroup of Z
5. Example (k>0): m: (S^1 , base) → (S^1 , base), m(base):≡base, m(loop):≡loop^k, gives kZ

6. Why not take the set bundle itself as subgroup? Indeed, alternatively:
Subgroup of (BG,sh)  is given by a (transitive) H : BG → Set, plus a point pt:H(sh),
such that Σz:BG H(z) is connected, underlying (sub)group is defined by (Σz:BG H(z),(sh,pt)).
Note that (sh,pt) = (sh,pt) consists of the symmetries of sh that keep pt:H(sh) fixed,
called the stabilizers of pt.

7. Example:  U: S^1 → Set: U(base):≡Zet, U(loop):≡s, point -17, gives trivial subgroup of Z
8. Example:  H: S^1 → Set: H(base):≡{0,..,k}, H(loop):≡ (s mod (k+1)), point 0, gives (k+1)Z.

9. Lagrange's Theorem: given group G and subgroup by (tr.) H : BG → Set, pt:H(sh),
 both G and H finite (??), then #(G) = #(H(sh)) x #(H).   (Is H(sh) the set of cosets ??)

10. Explanation finiteness. A type T is finite if there exists an n:N such that T is equivalent to the standard n-elt set (n unique, #(T), T set).  Group G is finite if UG is finite. G-set X is finite if X(sh) is finite (so are all X(z), with same #, as BG is connected).
 
11. Explanation H(sh). G-set H : BG → Set defines action H(g) : H(u) → H(v) for g: u = v,
denoted (g._), an equivalence. For x:H(sh), consider Cos(x):≡ {g:UG | g.pt = x}.
Then Cos(pt)=UH, and x=y if Cos(x) and Cos(y) overlap, and any g : Cos(x) determines an
equivalence (g_) between Cos(pt) and Cos(x), so any Cos(g.pt)=gCos(pt), indeed (g??).

12? Proof of Lagrange's Theorem: since the goal is a proposition, we may assume we have an equivalence to some standard n-elt set {0,...,n-1} for every finite set at hand. Then the
usual combinatorial argument (≈ 11 above) can be carried out in MLTT.

13? Infinite example:  H: S^1 → Set: H(base):≡{0,..,k}, H(loop):≡ (s mod k+1), pt=13, as above.
Now we have an equivalence from UG to H(base) x UH given by mod and div.
However, UG = H(sh) x UH cannot be obtained in general by elementary means (AC).
Yet we want to have a more general result that also works for infinite groups.

14. Lagrange's Construction: given group G and G-set H : BG → Set, pt:H(sh),
we have a function L_H: (Πx: H(sh) Σg:UG (g.pt = x))  →  UG = (H(sh) x UH).

Implementation.
Define [_] : UG → H(sh) by [g]:≡g.pt, then UG is sum of fibs of [_].
Thus UG is Σx: H(sh) Σg:UG x=g.pt is Σx:H(sh) ((sh,pt)=(sh,x)).
Finally, each ((sh,pt)=(sh,x)) is UH by f in L_H(f), using a=b → ((a=c)=(b=c)).

Remark: can be further generalised to pointed types BG: U and H: BG → U, 
but premise may be hard to fulfill.
